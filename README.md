# Site de Demonstração Prévia do PCG - 4° Sem

Este site é uma prévia do Projeto Curricular de Graduação, desenvolvido para ser totalmente acessível e inclusivo. Pensado para todos os tipos de público, ele é responsivo, garantindo ótima experiência em qualquer dispositivo. Além disso, possui recursos de acessibilidade, como suporte para pessoas com dislexia e interpretação em Libras.
